A full-stack typing performance platform that allows users to measure typing speed, accuracy, and errors in real time, with secure authentication and persistent score tracking.

# Key Features:

1. User registration and login using JWT authentication

2. Real-time WPM, accuracy, error, and progress calculation

3. Dynamic typing content fetched from an external Quotes API

4. Secure storage of user scores in MongoDB

5. Responsive UI built with Bootstrap 5

# Tech Stack:

1. Frontend: HTML, CSS, JavaScript, Bootstrap

2. Backend: Node.js, Express.js

3. Database: MongoDB, Mongoose

4. Auth & Security: JWT, bcrypt
